FinanceOS
First-Time Setup
1

Create a free Supabase account

Supabase is a free cloud database. Your financial data will be stored there and synced across all your team's phones.

→ Open supabase.com and sign up free
2

Create a new project

After signing up, click New Project. Give it a name like financeos. Choose any region (pick one close to Nigeria, e.g. eu-west-1). Set a strong database password and save it.

3

Create the database table

In your Supabase project, click SQL Editor in the left sidebar, then paste and run this SQL:

CREATE TABLE IF NOT EXISTS financeos_data ( id TEXT PRIMARY KEY, payload JSONB NOT NULL, updated_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE financeos_data ENABLE ROW LEVEL SECURITY; CREATE POLICY "public_access" ON financeos_data FOR ALL USING (true) WITH CHECK (true);

Click Run. You should see "Success".

4

Get your API keys

Click Project Settings → API in the left sidebar. You need:

  • Project URL — looks like https://xxxx.supabase.co
  • anon / public key — a long string starting with eyJ...

5

Save & connect

Click the button below. The app will connect to your Supabase database and you'll never need to set this up again on this device.
